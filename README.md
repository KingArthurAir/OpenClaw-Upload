# 염승환 투자클럽 랜딩페이지

최적화된 투자 클럽 랜딩페이지입니다.

## 기능
- 반응형 디자인 (모바일/데스크탑 호환)
- 사용자 관심사 선택 기능
- 사회적 증거 및 신뢰도 강화
- 법적 고지 및 컴플라이언스 준수

## 파일 구조
- `index.html` - 메인 랜딩페이지
- `static/css/style.css` - 스타일 시트
- `static/picture/tghr.jpg` - 프로필 이미지

## 사용법
1. 파일을 웹 서버에 업로드하거나
2. 로컬에서 직접 `index.html` 열기

## 최적화 내용
- JavaScript 함수명 오류 수정 (`cilck` → `handleClick`)
- 허위 데이터 제거 및 컴플라이언스 강화
- 모든 콘텐츠 중앙 정렬 보장
- 원본 색상 스킴 유지
- 이모지 통일 (`🚀` → `💬`)
